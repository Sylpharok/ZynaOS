# ZynaOS

## Español

ZynaOS es una distribución Linux basada en Pop!_OS 24.04 LTS, orientada al gaming y accesible para usuarios nuevos sin limitar a los avanzados.

## Sabores
- ZynaOS COSMIC (Core / Gaming)
- ZynaOS GNOME (Core / Gaming)
- ZynaOS Plasma Classic (Core / Gaming)
- ZynaOS Plasma Custom (Core / Gaming)
- ZynaOS Lite XFCE (Core / Gaming ligero)
- ZynaOS Lite LXQt (Core / Gaming ligero)

## Versiones
- 1.0 Spark
- 1.1 Pulse
- 2.0 Surge
